# alfrink-calendar

Webscraper for the [Alfrink calendar](https://www.alfrink.nl/agenda)

## Contributing

```bash
git clone https://github.com/stingalleman/alfrink-calendar
cd alfrink-calendar
npm install

node src/index.js # Run the code
```
